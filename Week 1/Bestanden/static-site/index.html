Kubernetes Cluster on Google Cloud

This application runs as a containerized workload inside a self-managed Kubernetes cluster hosted on Google Cloud — spanning three regions: Netherlands, Belgium, and the United Kingdom.

The image is built automatically via a GitHub Actions CI/CD pipeline and pushed to Docker Hub. From there it is deployed to the cluster using kubectl apply.
