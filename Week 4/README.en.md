# Week 4 - Helm & Identity and Access Management

## Topics

This week we work with **Helm**, the package manager for Kubernetes, and explore **Identity and Access Management (IAM)**. We learn how Helm charts work, how to deploy applications via a chart to a Kubernetes cluster, and how identity and access management works with Microsoft Entra ID, OAuth 2.0, OpenID Connect and SAML.

---

## Learning Goals

- Explore Helm as a package manager for Kubernetes
- Explore Identity Management (including Microsoft Entra ID)
- Explore Identity Providers (IdPs) like Okta
- Explore MDM / MAM solutions (including Microsoft Intune)
- Apply RBAC and Conditional Access
- Work with OAuth and other authentication protocols
- Explore OpenID Connect
- Experience SAML Single Sign-On (including GitHub Enterprise)
