# Week 4 - Helm & Identity and Access Management

## Onderwerpen

Deze week werken we met **Helm**, de package manager voor Kubernetes, en verkennen we **Identity and Access Management (IAM)**. We leren hoe Helm charts werken, hoe je applicaties via een chart uitrolt op een Kubernetes-cluster, en hoe identiteits- en toegangsbeheer werkt met Microsoft Entra ID, OAuth 2.0, OpenID Connect en SAML.

---

## Leerdoelen

- Helm verkennen als package manager voor Kubernetes
- Identity Management verkennen (o.a. Microsoft Entra ID)
- Identity Providers (IdPs) verkennen (o.a. Okta)
- MDM / MAM-oplossingen verkennen (o.a. Microsoft Intune)
- RBAC & Conditional Access toepassen
- Werken met OAuth en andere authenticatieprotocollen
- OpenID Connect verkennen
- SAML Single Sign-On ervaren (o.a. GitHub Enterprise)
