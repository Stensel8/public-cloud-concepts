De opdracht voor week 6 richt zich op het opzetten van een Proof of Concept op basis van een Cloud reference architectuur, specifiek het deel over microservices.

**Leerdoelen:**

- Hoe maak je van een monolithische applicatie een microservice?
- Hoe verbind je microservices via API Gateways (REST/HTTP/WebSockets)?
- Wat zijn microservices design patterns? (The Twelve-Factor App)

---

## 6.1 Microservices op GKE

![Architectuur: App users via Load Balancer en API Gateway naar microservices op Kubernetes](/docs/week-6/media/opdracht/image-001.avif)

1. Voer het lab [Migrating a Monolithic Website to Microservices on Google Kubernetes Engine](https://www.cloudskillsboost.google/focuses/11953) uit en lever het bewijs van voltooiing in als onderdeel van je portfolio. Als je het lab niet volledig afrondt, laat dan screenshots zien van de fasen die je wel hebt doorlopen.

2. Leg uit hoe je een API Gateway kunt toevoegen aan de oplossing die je in het microservices-lab hebt gebouwd. Bestudeer en oefen met [Secure traffic to a service with Google Cloud Console](https://cloud.google.com/api-gateway/docs/secure-traffic-console).
