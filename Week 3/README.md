# Week 3 - Blue-Green Deployments & Artifact Registry

## Onderwerpen

Deze week implementeren we een **Blue-Green deployment** voor een applicatie via **Google Artifact Registry** als alternatief voor Docker Hub. We koppelen een CI/CD-pipeline die automatisch een nieuw image bouwt en uitrolt bij elke codechanging.
