# Week 3 - Blue-Green Deployments & Artifact Registry

## Onderwerpen

Deze week implementeren we een **Blue-Green deployment** voor een applicatie via **Google Artifact Registry** als alternatief voor Docker Hub. We koppelen een CI/CD-pipeline die automatisch een nieuw image bouwt en uitrolt bij elke codechanging.

---

## Leerdoelen

- Blue-Green deployment opzetten en begrijpen
- Google Artifact Registry gebruiken als container registry
- CI/CD pipeline inrichten met GitHub Actions voor deployment naar GKE
- Wisselen tussen Blue en Green versies via een Kubernetes Service
- Alternatieven voor GitHub Actions kennen: Argo CD en Flux CD
