# Week 3 - Blue-Green Deployments & Artifact Registry

## Topics

This week we set up a **Blue-Green deployment** for an application using **Google Artifact Registry** as an alternative to Docker Hub. We connect a CI/CD pipeline that automatically builds and deploys a new image on every code change.

---

## Learning Goals

- Set up and understand Blue-Green deployments
- Use Google Artifact Registry as a container registry
- Set up a CI/CD pipeline with GitHub Actions for deployment to GKE
- Switch between Blue and Green versions via a Kubernetes Service
- Know alternatives to GitHub Actions like Argo CD and Flux CD
